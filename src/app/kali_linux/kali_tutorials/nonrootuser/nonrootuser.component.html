Kali Linux is a Debian based Linux distribution aimed at advanced penetrating testing and security audit ,Kali Linux comes with post-installation of software which is used for security purposes, forensic science, and reverse engineering

Which command are used to create new user

We all are aware of most popular command useradd some time as system administrator asked to create a new user on Linux with some specific limitation that time we can create a new user with useradd command

Perks of having Non Root Account on Kali

  • mitigate the problem of system vulnerability
  • user can perform decided task

Steps By Steps Guide of Create New Non Root Account

Steps:1
Open the Terminal and follow given instruction

Step:2
entered following command

$ useradd -m alpha(username)
-m is here used for create account under home directory


Step:3

Next we have to set Password for account we have created

$ passwd alpha

Step:4
Now we have to add this user account to sudo group so in future there is need raise they can use for instance any software installation etc

$ usermod -a -G sudo alpha

Step:5
Now we will change login shell for new created user

$ chsh -s /bin/bash alpha

That's all we have created new non root user account
