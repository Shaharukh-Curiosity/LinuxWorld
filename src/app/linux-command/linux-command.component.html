Top 10 Linux Basic Command

Created By: Shaharukh Parmar - 01, Feb,2020


1:cat command
cat is used to view the content of the file or concatenates files, create a new file

Syntax
  $ cat filename 

2:pwd command
display the current working directory

Syntax
  $ pwd

3:cd command
move from one directory to another directory

Syntax
  $ cd Desktop

4:mkdir command
is used to create the new directory

Syntax
  $ mkdir filename

5:rmdir command
is used to remove the created directory

Syntax
  $ rmdir directoryname

6:rm -r directory command
remove the file and directory recursively

Syntax
  $ rm -r directoryname

7:touch command
is used to create a new file

Syntax
  $ touch filename

8:cp command
is used to copy the files

Syntax
  $ cp sourcefile destinationfile
cp hello.txt /Desktop

9:hostname command
display the host machine name

Syntax
  $ hostname

10:uptime command
is used to show how long system had been running, number of logged users

Syntax
  $ uptime
