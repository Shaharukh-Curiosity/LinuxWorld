Storage is basic need of every computer , File System is a logical method of organizing and storing abundance amount of data in a way that make it easy to manage the files on machine ,Linux Provide the tools to manage disk partition with help of fdisk , you can view how much partition has been done on disk by running the utility fdisk with -l option

Now time to run fdisk with -l to check the partition table on disk
- some berif information has been printed by fdisk -l
Device:Name of partition
Start & End: Starting to ending address of sector
Blocks:Total size of the block
FileSystem:partition contain which type of the file, it might be (ext4,ext2,vfat) - you can also check the size of hard drive with the help of parted command with -l option along sudo privilege
