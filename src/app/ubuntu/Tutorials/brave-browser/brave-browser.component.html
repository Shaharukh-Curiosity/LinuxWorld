Steps By Steps Guide to Install The Brave

 

Step 1 : 

       - Open the terminal Via (ALT+CTRL+T) or by searching with the help of App Launcher

         

Steps 2:

      -    in order to install the Brave on the machine before that we must have to install the curl on our machine(by default on most Linux distros it comes pre-installed)



Steps 3: 

    

  •  Now then add Repository to your system by typing the following commands

 

           $  curl -s https://brave-browser-apt-release.s3.brave.com/brave-core.asc | sudo apt-key --keyring /etc/apt/trusted.gpg.d/brave-browser-release.gpg add 




Steps 4:

 -Now time is up to add a repository in the sources. list.d(directory)

  

  $ echo "deb [arch=amd64] https://brave-browser-apt-release.s3.brave.com/ stable main" | sudo   tee /etc/apt/sources.list.d/brave-browser-release.list




Steps: 5 

  • Finally, we have reached the mark where only one steps remain to install the brave web browser on our system
  • $ sudo apt-get update && sudo apt-get install brave-browser
  • It will ask you to continue or abort the installation in form of (y/n) (to continue to press the Y and to abort press the N)
