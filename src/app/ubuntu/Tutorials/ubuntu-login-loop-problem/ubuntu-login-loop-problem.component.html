Solve the problem of ubuntu 20.04(Focal Fossa)login loop problem

Written By: Shaharukh Parmar: 13,Dec 2020



-

- Some of you might have installed fresh ubuntu 20.04 or maybe someone may have upgraded from their old Linux distros and now you are facing a login loop problem so let move ahead and try to solve it without taking much time

 

Step1:

- Connect your machine with an active internet connection so your package can be installed (if you are accessing the internet connection through a hotspot, you have to enable tethering option to continue access the internet on your machine

 

Step2:

- Now press the alt+ctrl+f3 same time to open the command line tools to move ahead

 

Step3:

- Now  enter the below commands 




- Guys don't forget to login into system before execute below commands(in your case username(computer name)& password

- Sudo apt update

- Sudo apt upgrade

-Reboot

 

Step4:

  - Now your machine will be rebooting, once the machine starts displaying a login screen now enter the login credentials to enter into the system
